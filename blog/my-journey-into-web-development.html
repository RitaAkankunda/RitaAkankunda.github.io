My Journey into Web Development

From zero knowledge to building modern web applications.

February 3, 2026

As a Computer Science student, my journey into web development began with curiosity about creating interactive and user-friendly websites. I started with HTML, CSS, and JavaScript, then moved to frameworks like React to build dynamic applications.

Learning full-stack development with Node.js, Express, and MongoDB allowed me to create complete web solutions. Through projects like Campus Hustle, I've applied these skills to build meaningful applications that solve real-world problems.

Web development has taught me the importance of responsive design, user experience, and clean code. I'm passionate about creating digital solutions that are accessible and impactful. The tech industry offers the freedom I crave – remote work, work-life balance, and the ability to pursue other interests while doing what I love.
