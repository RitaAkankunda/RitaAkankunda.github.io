## Featured Projects

### 1. Campus Hustle
A platform designed to solve the problem of cluttered WhatsApp groups at universities. Campus Hustle enables student entrepreneurs to post items for sale in one dedicated place, reducing chat traffic and keeping important notices visible.

**Tech Stack:** JavaScript, TypeScript, Node.js, Express, MongoDB  
**Repository:** [View on GitHub](https://github.com/RitaAkankunda/campus-hustle)  
**Live Demo:** [campus-hustle.netlify.app](https://campus-hustle.netlify.app/)

### 2. Dembe Driving School
A modern, responsive website for a professional driving school in Uganda. Features a beautiful green-themed design with realistic vehicle animations and comprehensive course information.

**Tech Stack:** React, TypeScript, Tailwind CSS  
**Repository:** [View on GitHub](https://github.com/RitaAkankunda/dembedrivingschool)  
**Live Demo:** [View Live](https://ritaakankunda.github.io/dembedrivingschool/)
